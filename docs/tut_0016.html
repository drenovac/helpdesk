Verifying your EMPDUP file

This document outlines how to download the ATO utility which allows you to verify your EMPDUP file prior to sending it into the ATO for EOY submission.

Introduction

The ATO has made available a utility they call the Electronic Commerce Interface (ECI), the details of which can be found at this link. We encourage you to download the utility and test your EMPDUP files through this program. This link has detailed instructions on how to download and install the software, however, our support staff are very happy to assist you in case you have problems.
