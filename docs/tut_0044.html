Doing Payroll I have come across a issue...

Powerforce is not calculating a public holiday that falls on Monday for a shift coming across from Sunday night...

What I mean :
Sunday night shift: 1800-0600
1800-0000: Sunday loading
0001-0600: should be pholiday loading...

The way it is behaving at present is it is taking that 0001-0600 as Sunday loading when Monday is public holiday
And when Monday is not pholiday it calculates 0001-0600 as night shift which is fine...

So the issue is only when Monday is a public holiday

Explanation

In order to resolve this issue, we have followed the process of a recent helpdesk ticket (verify PHOL helpdesk ticket). The issue descibed above however has one more small twist, and as per the description of the problem of the shift flowing Sunday into a Public Holiday, the configuration is a special setting (below). Check the box as shown, and then re-cost your shift. You will see that the loadings flow as per your requirement noted.
