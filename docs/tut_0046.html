Overview

This document outlines steps to print any Powerforce report in formats other than those provided for by Powerforce. Or, if indeed the print options in Powerforce do not provide sufficient flexibility.

See Also

Helpdesk index pre-2013 technotes

Explanation

In the case of generic report, you are most often provided with the option to print to .pdf or email. In case these options do not work for you, follow these few steps to work another option to get your reports out in the format that you require.

 

All Powerforce reports by default go to screen for preview. Once you are in the preview window you will be provided the following options to print with (lower left hand corner of preview window).


If you selecct the suitcase (3rd icon from left), you are then provided with further options to select your output format, as the following image shows:

The most common options to use:

  • pdf - to create a .pdf formatted report
  • .csv - to create an output suitable for excel
  • .html - the preferred option to create an output suitable for excel. This option will create a file which will bring across the format options of your report unlike the .csv which is purely for data upload.

 

Under "File Name", use the usual Windows idioms to select the location of where to store your file. If you do not type the pathname manually, press on the 3 dots (...) following to prompt you for the location and file name. Once you've entered a valid file name, the Ok button will become usable and you may then save the document.
