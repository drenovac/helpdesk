Explanation

In the case of client ABC, the export function calls for a mapping of internal Powerforce AI codes to specific award based span codes (as used in Micropay).

 

To create the configuration, work backwards from Micropay into Powerforce.

In the example shown here - in Micropay the span code is called "SYDMARF C".
Create a Generic Reference entry as shown below with the exact id as is required in Micropay (dropdown shows other span codes).


The second part is to create an Allowance whose id ends in "SPAN".

In this example below, the allowance code is called "SYDMSPAN", and its definition is shown below.

Finally on the classification code definition, link the allowance code to the award link code (NSN in the example below, infering that the award rule "NSW-SPAN" as such a code defined).

So, what you have is the classification pointing to an allowance code ending in "SPAN", which then in its own definition has an "Export Cross Reference" of the generic reference code ("SYDMARF C" the actual Micropay span code).
