New function - add menu options

In order to provide for the auto-update of Powerforce menu options, a new option has been added to the user administration screen - where menu options are set.

See Also

Technote index

Overview

You need to ensure that you are at patch release e096 at least.

Using Administration / Users, click the button as show, and ensure that the system has provided an update of "items added"

 

Then without having to re-enter the window, select the user(s) to whom you wish to add the new option, and follow the regular procedures to add the new menu option to your menus.
