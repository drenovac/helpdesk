I have attempted several times to upload images to individual guards profiles in powerforce and I have not had any success; can you please assist?

Explanation

The configuration of the images directory for your employees (on a per company basis), is managed via the "Company System Defaults" settings.

What you notice in the above configuration is that both the employee and client gallery locations point to the same sub-directory (looks like client gallery in this instance).

Changing the 'cgallery' to 'gallery' (or the absolute location of where you store your photo images), in the Personel Gallery position will correct the issue.

A quick confirmation with the gallery button of the employee details screen confirms that the location is now correctly set up.

 


Many clients are going through the transition of migrating to our current 'new' version of Powerforce (version 8), and the configuration for the version 8 is just slightly different. Version 8 Powerforce sits at the same directory level as your current production version, with version 8 pointing back into the current version for data and all other settings.
